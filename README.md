## 🧬 5. IVF Pregnancy Success Prediction

난임 환자의 시험관 아기(IVF) 시술 성공 여부 예측 AI 프로젝트

보조생식술(ART) 임상 데이터를 활용하여 환자의 임신 성공 가능성을 예측하는 머신러닝 모델을 개발했습니다.

문제 배경

반복 시술로 인한 경제적/심리적 부담 존재

목표

환자의 시술 데이터를 기반으로 개인 맞춤형 임신 성공 확률을 예측하는 의사결정 보조 모델 구축

시술 당시 나이

배아 관련 정보
